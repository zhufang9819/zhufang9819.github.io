## 简介
最大化最小值和最小化最大值是同一类问题，比如我们要对区间进行分割，设分割后得到的区间的元素之和中最大的那个是 $g$，而我们要使得这个 $g$ 尽量的小，这就是一个最小化最大值的问题。
我们可以用二分搜索来解决这个问题，搜索某个值 $d$ 使得 $d$ 比分割的所有子区间的和还要大，在此基础上我们要最小化 $d$，但是本文我们并不考虑这样的问题整体上怎么解决，而是关注它的一部分，
即候选解区间的二分搜索写法问题。

## 复习
二分搜索就是对**有序区间**的检索算法，每次都访问区间的中点，然后根据结果选择舍弃该区间的某一半，直到得到最终的解为止，至于舍去哪一半区间，这个要结合具体的任务来看，我们给一个典型的区间检索的二分算法

{% highlight cpp %}
/*  
    数组 arr 是递增有序的
    找到待找元素返回下标(1 到 n)
    否则返回0
*/

int arr[n];
int binarySearch(int x, int lo, int hi, int k) {
    int mid;
    while (lo <= hi) {
        mid = lo + (hi-lo)/2;
        if (arr[x] == k) return mid;
        else if (arr[x] > k) hi = mid-1;
        else lo = mid+1;
    }
    return 0;
}

{% endhighlight %}

二分检索的算法时间复杂度是 $\mathcal{O}(\log n)$
